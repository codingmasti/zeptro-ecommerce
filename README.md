# 🛍️ Zeptro - Multi-Category E-Commerce Web Application

Zeptro is a modern and responsive multi-category e-commerce web application built with React.js and Tailwind CSS. It provides a seamless online shopping experience where users can browse products from different categories, search items, manage their cart, add delivery addresses, choose payment methods, and place orders.

The application is designed with a clean UI, responsive layout, and reusable React components to ensure a smooth shopping experience across desktop, tablet, and mobile devices.
